# Saree_Retail_Ledger
A simple and elegant web application to manage customer transactions and pending payments for saree businesses.

# Overview
The application enables small business owners to maintain a digital ledger of their customers, including outstanding balances. It focuses on simplicity, usability, and quick access to essential information.

# Features
- Add and manage customer records
- Search and view customer details
- Track pending payment amounts
- Dashboard displaying total customers and total pending balance
- Clean and minimal user interface

# About This Project
The Saree Ledger is a custom digital bookkeeping solution designed to modernize retail clothing businesses by transitioning them from traditional paper records to a secure, automated system.

- ### My Contribution:
    I engineered this end-to-end solution from scratch as a full-stack developer, utilizing Java and Spring Boot. My work spanned designing the responsive frontend interface, architecting the backend transaction logic, and managing secure data flow.

- ### Cloud Infrastructure:
    Successfully integrated the application with a live cloud database using Aiven and managed the full web deployment via Render.

- ### Real-World Impact:
    The application has transitioned from a development project into a live, production-ready business tool. It is currently being used daily by my father on his mobile phone to seamlessly manage his shop's customer ledgers, track apparel categories, and automate pending balance calculations.
